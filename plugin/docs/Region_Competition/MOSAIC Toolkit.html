Installing MOSAIC Toolkit

MOSAIC Toolkit is a collection of plugins, for image segmentation, particles tracking, Interaction analyses, and others ...

Installing MOSAIC Toolkit

To install MOSAIC toolkit requires only to put the jar file inside the plugins folder of ImageJ or ImageJ2, the package in general is tested on the most common ImageJ distributions

Usage

Open ImageJ or ImageJ2, you will find all MOSAIC plugings under Plugins -> MOSAIC menu
